# Supported Versions

Nova Sphere follows Semantic Versioning (SemVer). 
Security and bug fixes are backported based on the following support schedule:

| Version | Status | Security Support Ends | Bug Fix Support Ends |
| :--- | :--- | :--- | :--- |
| **v2.0.x** | **Active Support** | Dec 2028 | Dec 2027 |
| **v1.x** | **End of Life** | *Ended* | *Ended* |

## v2.0.x Maintenance Policy
- The `main` branch tracks the next minor/major release.
- Hotfixes are applied to `release/v2.0.x` and tagged sequentially (e.g., `v2.0.1`, `v2.0.2`).
